DEPLOYMENT OPTIONS
1. GitHub Pages:
   - Upload all files to the root of your repository.
   - Make sure index.html is in the root.
   - In GitHub Pages settings, publish from the main branch / root.

2. Existing hosting:
   - Upload the full folder contents to the site root via your hosting panel or FTP.
