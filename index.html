FAQ

Straight answers for project teams.

Key questions taken from the current BodenRyan website, tightened up for a cleaner user experience.

Can you change the number of images captured each day?

Yes. Scheduling can be adjusted remotely depending on the project requirements.

What is the image viewer?

It is a web-based portal that allows you to view and download project images throughout the build.

Do you provide the data connection?

Yes. Camera systems can be connected via 4G, Wi-Fi or Ethernet as required.

How often are images taken?

This depends on the project. Typical schedules range from burst captures to intervals every 5–15 minutes.

Can I install the system myself?

No. Installation is handled professionally to ensure safety, reliability and the correct framing.

Where do you operate?

BodenRyan is based in Nottingham and works across the UK.
